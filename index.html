Maksimal ukuran file: 10MB

Template Format
  • • Kolom wajib: NIK, NAMA, GOL, NO KAMAR, LOKASI HUNIAN
  • • Format NIK: Bebas (huruf dan angka)
  • • GOL: Bebas (tidak ada batasan)
  • • Format tanggal: YYYY-MM-DD
  • • Format status: TERISI, TERSEDIA, RUSAK, DIPERBAIKI
  • • Lokasi Hunian: P2 SURYA RESIDANCE, MENTARI, PELANGI
  • • Fasilitas: CR, NR
  • • Perusahaan: PT. HJF, PT. INKA, PT. KAI, VENDOR LAINNYA
  • • Floor: Lantai Bawah, Lantai Atas (atau kosongkan)
Catatan Penting
  • • Data duplikat berdasarkan NIK akan diupdate
  • • Data akan divalidasi sebelum diimport
  • • Backup disarankan sebelum import massal
  • • Lokasi hunian harus sesuai dengan pilihan
  • • NIK bisa diisi dengan kombinasi huruf/angka apapun
  • • Floor hanya boleh: Lantai Bawah, Lantai Atas, atau kosong
