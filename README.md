# Aplikasi Data Hunian Mess Karyawan

Aplikasi web untuk mengelola data hunian mess karyawan dengan kapasitas hingga 20.000 karyawan.

## Fitur Utama

1. **Dashboard** - Statistik dan visualisasi data
2. **Manajemen Karyawan** - CRUD data karyawan
3. **Manajemen Hunian** - Monitoring kamar mess
4. **Import/Export Excel** - Import data dari file Excel
5. **Pencarian & Filter** - Cari data dengan berbagai kriteria
6. **Laporan** - Generate laporan hunian
7. **PWA Support** - Dapat diinstall di perangkat mobile

## Teknologi

- HTML5, CSS3, JavaScript (Vanilla)
- Bootstrap 5.3 untuk UI
- Chart.js untuk visualisasi
- SheetJS (XLSX) untuk import/export Excel
- Supabase untuk backend (optional)
- PWA untuk offline capability
